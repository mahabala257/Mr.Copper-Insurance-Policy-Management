# Insurance Policy Management System

A modern **Insurance Policy Management System** built using JavaScript, HTML, and CSS.
This project simulates a real-world insurance workflow including **policy creation, premium tracking, claims processing, and underwriting**.

---

## Features

- Policy Management (Create, Renew, Lapse, Surrender)
- Premium Tracking (Paid, Overdue, Grace Period)
- Claim Processing (Submit → Review → Approve/Reject → Settle)
- Underwriting System with Risk Assessment
- Policyholder Management
- Dashboard with real-time statistics

---

## No Hardcoding (Key Highlight)

Unlike basic projects, this system:

- Does NOT rely on fixed/static data
- Allows dynamic data creation through UI
- Can be extended with Local Storage / Database
- Designed to simulate real-world application behavior

## How It Works

The system is built using object-oriented design:

- **Policy Class** → Handles lifecycle (Active, Lapsed, Renewed)
- **Claim Class** → Handles claim workflow & settlement
- **Premium Class** → Tracks payment status
- **Underwriter Class** → Risk analysis & approval decisions
- **InsuranceStore** → Central data management

---

## Core Concepts Used

- Object-Oriented Programming (OOP)
- State Management using Classes
- Event-driven UI (JavaScript)
- Modular Design
- Basic Risk Analysis Logic
